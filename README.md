# Google Cleaner

- old style links
- moves the top bar (All, Videos, News...) to the sidebar
- hides "rich search content" (Top stories...)
- adds toggle results in english/all languages

Get as an [userscript][1] or an [firefox][2] or [chrome][3] extension.
